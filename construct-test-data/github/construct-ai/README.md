# Construct AI

An AI-powered development assistant that helps with software engineering tasks.

## Features

- Code generation and refactoring
- Debugging assistance
- Architecture planning
- Test creation
- Documentation generation

## Installation

```bash
npm install @construct/ai
```

## Usage

```javascript
const { Construct } = require('@construct/ai');
const ai = new Construct();
ai.helpWithTask("Create a REST API endpoint");
```
